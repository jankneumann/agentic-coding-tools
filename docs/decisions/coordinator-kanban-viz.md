## 2026-05-15 — 2026-05-22-add-coordinator-kanban-viz

### Phase: Plan

**SSE (one-way) for live updates, not WebSocket** — Live data flows server→client only. SSE reuses HTTP auth, traverses proxies, has trivial client surface (EventSource). WebSocket framing/ping-pong adds no UX value on a read-mostly board. Polling fallback engages transparently on EventSource error.
